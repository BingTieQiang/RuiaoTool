本项目是个人开发所用的一套常用框架，里面包含OkHttp的网络请求封装、tablayout的自定义，SmartRefreshlayout下拉上拉，activity右滑退出，以及各种布局回弹效果和android的版本适配（这些不是主要目的，主要。。。请看下面👇）。

这里主要介绍近期项目所需的表格和折线以及二维表格，有需求的同学可以不妨借鉴 😆。
